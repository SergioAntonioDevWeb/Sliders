# Proyecto Slider - Versión 1.0

Este es un proyecto de Slider (carrusel de imágenes) interactivo y totalmente responsivo, construido utilizando tecnologías web estándar (HTML5, CSS3 y JavaScript Vanilla).

## 🚀 Características

- **Interactividad**: Navegación fluida entre diapositivas mediante flechas laterales.
- **Diseño Responsivo**: Adaptado para dispositivos móviles, tablets y escritorio mediante Media Queries.
- **Optimización**: Lógica de JavaScript optimizada para el manejo de clases y transiciones.
- **Tipografía Moderna**: Uso de la fuente "Raleway" de Google Fonts.
- **Animaciones**: Transiciones suaves de opacidad al cambiar de diapositiva.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica del contenido.
- **CSS3**: Estilos personalizados, Flexbox, CSS Grid y diseño responsivo.
- **JavaScript (ES6+)**: Lógica funcional para el cambio de diapositivas y manejo de eventos.

## 📂 Estructura del Proyecto

```text
/
├── assets/          # Imágenes y recursos visuales (iconos, fotos de perfil)
├── estilos.css      # Hoja de estilos principal con diseño responsivo
├── index.html       # Estructura principal del slider
├── sliders.js       # Lógica de programación del slider
└── README.md        # Documentación del proyecto
```

## 💻 Instalación y Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/SergioAntonioDevWeb/Sliders.git
   ```
2. Navega a la carpeta del proyecto.
3. Abre el archivo `index.html` en tu navegador preferido.
